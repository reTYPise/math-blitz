# math-blitz

**Скорость мышления** — веб-приложение для тренировки устного счёта. Минималистичный интерфейс, несколько режимов игры и способов ввода ответа.

![math-blitz](https://img.shields.io/badge/math--blitz-скорость%20мышления-e8ff47?style=for-the-badge&labelColor=0a0a0f)

## Возможности

### Операции
- **Умножение** (×)
- **Деление** (÷)
- **Сложение** (+)
- **Вычитание** (−)

Можно выбрать одну или несколько операций одновременно.

### Диапазон чисел
- 1–10, 1–12, 1–20, 1–25, 1–50, 1–100

### Режимы игры

| Режим | Описание |
|-------|----------|
| **Классика** | N вопросов (10, 20, 30, 50, 100). Прогресс-бар показывает оставшиеся вопросы. |
| **На время** | За отведённое время (30 сек – 5 мин) решить как можно больше примеров. |
| **Выживание** | Максимум правильных ответов за лимит времени. Ошибки не заканчивают игру. |

### Ввод ответа
- **⌨ Клавиатура** — ввод цифр с клавиатуры, Enter для ответа
- **☰ Варианты** — выбор из 4 вариантов ответа
- **📱 Экран** — виртуальная клавиатура для сенсорных устройств

### Дополнительно
- Серия правильных ответов (streak)
- Звуковая обратная связь (правильно / ошибка / предупреждение / время вышло)
- Визуальные эффекты (вспышка при ответе, анимации)
- Таймер с предупреждениями при 50% и 20% оставшегося времени

## Запуск

Откройте `index.html` в браузере. Внешние зависимости отсутствуют (кроме Google Fonts).

```bash
# Или через локальный сервер
npx serve .
# или
python -m http.server 8000
```

## Технологии

- **HTML5** — разметка
- **CSS3** — стили, CSS-переменные, анимации
- **Vanilla JavaScript** — логика игры
- **Web Audio API** — звуковые эффекты
- **Google Fonts** — Space Mono, Syne

# math-blitz

**Скорость мышления** — веб-тренажёр устного счёта с дашбордом прогресса. Статический сайт на GitHub Pages.

**Сайт:** https://retypise.github.io/math-blitz/

![math-blitz](https://img.shields.io/badge/math--blitz-скорость%20мышления-e8ff47?style=for-the-badge&labelColor=0a0a0f)

## Возможности

### Операции
- **Умножение** (×), **деление** (÷), **сложение** (+), **вычитание** (−)
- Можно выбрать одну или несколько операций одновременно

### Диапазон чисел
- 1–10, 1–12, 1–20, 1–25, 1–50, 1–100
- Фокус на конкретное число или любые числа

### Режимы игры

| Режим | Описание |
|-------|----------|
| **Классика** | N вопросов (10–100) |
| **На время** | Максимум за 30 сек – 5 мин |
| **Выживание** | Максимум правильных за лимит времени |

### Ввод ответа
- **⌨ Клавиатура** — цифры + Enter
- **☰ Варианты** — выбор из 4 ответов
- **📱 Экран** — виртуальная клавиатура

### Дашборд
- Серия дней, KPI (сессии, точность, сегодня)
- Активность за 7 дней
- Breakdown по операциям
- Последние сессии
- Быстрый старт из шапки

## Структура

| Файл | Назначение |
|------|------------|
| `index.html` | Разметка: дашборд, игра, экран счёта |
| `css/styles.css` | Стили, mobile-first, ширина до 1240px |
| `js/app.js` | Игра, вопросы, UI, дашборд |
| `js/db.js` | SQLite (sql.js) → `localStorage` |
| `js/vendor/` | sql-wasm.js + sql-wasm.wasm |
| `agents.md` | Контекст для AI-агентов |

База создаётся в браузере каждого пользователя (ключ `math-blitz-sqlite-v1`). Сервер не нужен.

## Локальный запуск

```bash
npx serve -l 3000
```

## GitHub Pages

1. **Settings → Pages → Source:** GitHub Actions (или branch `main`, folder `/`)
2. Пуш в `main` — workflow `.github/workflows/pages.yml` публикует сайт

URL: `https://retypise.github.io/math-blitz/`

## Технологии

- HTML5, CSS3 (Grid, переменные, анимации)
- Vanilla JavaScript (ES6+)
- sql.js (SQLite в браузере)
- Web Audio API
- Google Fonts — Space Mono, Syne
